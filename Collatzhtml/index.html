About This Project

This interactive Collatz Explorer was created by Shaimaa Soltan. It visualizes inverse Collatz trees, odd-child families, v₂ factorization, and dynamic spiral mappings.

Built with JavaScript, HTML5 Canvas, vis-network, and designed with the help of Microsoft Copilot.
